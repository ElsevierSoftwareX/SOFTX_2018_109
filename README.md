# Niget

Software tool for detailed analysis of instrumented indentation measurements


## Prerequisities

- GTK2
- Gwyddion
- CMake
- C compiler (plus Fortran compiler recommended)


## Installation (Linux)

- mkdir build
- cd build
- cmake ..
- make


## Running (Linux)

The 'niget' executable located in the build directory
